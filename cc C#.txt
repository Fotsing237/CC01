Le projet consiste donc à:
- Créer une application pour produit les cartes d'étudiant comme ci-dessous.
- Un étudiant appartient à une école.
- Un formulaire pour créer, modifier, supprimer et rechercher, imprimer la carte de l'étudiant avec les informations nécessaires 
- Un formulaire pour créer, modifier,  supprimer, rechercher des écoles avec les informations nécessaires (un seul logo suffira)
- Un report pour imprimer les cartes
- Un formulaire parent pour contrôler la navigation sur l'application 
- Le stockage de données en json dans un répertoire DATA courant au projet Windows Forms.
- Le code barre sur la carte est le matricule encodé de l'étudiant et stocké sous la forme d'une image.
